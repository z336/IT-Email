Updates

Office 365

  • Outlook Web App - Select text to quote in replies. Select some text, click quote, and Outlook Web App creates a message with the text quoted. Works in the browser, Outlook for Android and iOS
  • Office 2016 for Windows - add links to recent files in your documents, spreadsheets or presentations
  • The latest version of Office 2016 for Windows is 1704 Build 8067.2115
  • The latest version of Office 2016 for Mac is 15.34.0 (17051500)
  • Office for Windows should auto-update but if it hasn't you can update it yourself. Go to File--> Account in any Office program and click the Update Options button.
  • Office for Mac can be updated by using Microsoft AutoUpdate
  • If you have trouble updating to v1704 please contact the helpdesk.
