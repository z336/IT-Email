Open office experiments

  • IT occupied Seminar Room West during the week of May 22-26 in order to experiment with working in an open office environment. Overall, we found it to be a great way to increase communication while completing tasks. It broke down walls, literally and figuratively, between members of the team. Knowledge transfer became a daily routine. It takes a little extra concentration in order to focus on solo work, but the overall goal is also to limit how much work we have to do on our own.

Office 365

  • Outlook Web App - Select text to quote in replies. Select some text, click quote, and Outlook Web App creates a message with the text quoted. Works in the browser, Outlook for Android and iOS.
  • Office 2016 for Windows - add links to recent files in your documents, spreadsheets or presentations.
  • The latest version of Office 2016 for Windows is 1705 Build 8021.2025
  • Office 2016 for Windows v1705 has a bug in Skype that causes Outlook to produce error messages. If you've been bitten by this bug and want to downgrade to the latest stable version please contact the helpdesk.
  • The latest version of Office 2016 for Mac is 15.34.0 (17051500)
  • Office for Mac can be updated by using Microsoft AutoUpdate.
  • The Microsoft Planner app is available for iOS and Android.
